Policies and Licenses Register - BETA

A Linked Data API delivering Registers of ontologically characterised Policies which includes Licenses.

This API delivers a series of Registers of different classes of things (Policies, Licenses etc.) and also web forms and underlying API endpoints for creating new class instances. All of the endpoints and functions of the API are listed below.

API Contents

Content Registers

Register Endpoint
Register of Registers reg/
Policies policy/
Licenses license/

Functions

Function Endpoint
Create a License function/new-license
User Login function/login

New functions like 'Create a Policy' will be added soon!
