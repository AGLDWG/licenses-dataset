About

This website is an Application Programming Interface (API), which means it is a machine-readable resource, that delivers instances of and the ability to create Policies which are "statements about the usage of content and services" (from the ODRL Information Model 2.2). Policies here are formally defined by the ODRL Version 2.2. Ontology and include Licenses as a specialised type of Policy such as the well-known Creative Commons series of Licenses often used to license data internationally and in particular by the Australian government.

How the data is produced

Most of all the well-known Licenses listed in this API (see the Licenses Register) were initially characterised by work done for the Licentia tool. The characterisations of the licenses have also since been added to by the creators of this API.

Characterisations of Policies and Licenses beyond those from Licentia have been added by the creators of this tool and allowed contributors. Individual Policies and Licenses have their creators indicated in their metadata.

If you want to add Policies and Licenses...

Additions to this system are allowed from members of the Australian Government Linked Data Working Group only thus if you wish to add Policies and Licenses, please join the Working Group!

Members of the Working Group will be able to log in to this system and create new Policies and Licenses via the login page.

Status of the tool

This tool was first produced in May, 2018, and for the sort while following is in Beta testing mode as indicated on the home page. This Beta designation will be removed when the tool is in full production.
